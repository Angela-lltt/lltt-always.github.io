readyState不同值的表示

  • 0(创建对象)：(XMLHttpRequest)对象已经创建，但还没有调用open()方法。
  • 1(初始化请求)：已经调用open() 方法，但尚未发送请求。
  • 2(发送请求)：请求已经发送完成。
  • 3(解析响应)：可以接收到部分响应数据。
  • 4(完成)：已经接收到了全部数据，并且连接已经关闭。
